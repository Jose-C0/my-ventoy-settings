# my-ventoy-settings

Este repositorio contiene la configuración y los recursos necesarios para personalizar una instalación de Ventoy.

## Estructura básica del repositorio

- `ventoy/ventoy.json`: configuración principal de Ventoy.
- `ventoy/ISO/`: imágenes ISO y archivos de arranque que se usan con Ventoy.
- `ventoy/themes/`: temas personalizados para la interfaz gráfica de Ventoy.

## Uso recomendado

1. Copia la carpeta `ventoy` dentro de la llave USB o la carpeta raíz de Ventoy.
2. Asegura que `ventoy/ventoy.json` esté ubicado en la carpeta `ventoy` junto a `ventoy/ISO/` y `ventoy/themes/`.
3. Arranca Ventoy y verifica que los temas y las ISO se detecten correctamente.
